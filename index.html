CSV 格式說明

第一列為標題列，需依序包含以下欄位：

欄位 說明
date 交易日期，格式 YYYY-MM-DD
symbol 股票代號，例如 2330 或 AAPL
name 顯示名稱，可空白
action buy（買進）或 sell（賣出）
quantity 股數，必須大於 0
price 單股成交價，需大於等於 0（台股為 TWD，美股為 USD，依匯入時選擇的市場決定）
fee 手續費，可空白（視為 0）

範例：2024-01-10,2330,台積電,buy,300,560,20

格式不符的列會被略過並標示行號與原因，其餘正確的列仍會正常匯入。
